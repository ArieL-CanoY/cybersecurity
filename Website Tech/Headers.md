# Request Headers
```python
>> Accept: Specifies the media types the client is willing to receive.
	Example: "Accept: text/html, application/json"

>> Accept-Encoding: Specifies the content encoding (like compression) the client can handle
	Example: "Accept-Encoding: gzip, deflate"

>> Accept-Language: Specifies the preferred languages for the response.
	Example: "Accept-Language: en-US, en;q=0.9"

>> Cache-Control: Directives for caching mechanism in both requests and responses.
	Example: "Cache-Control: no-cache"

>> Content-Type: Indicates the media type of the request body.
	Example: "Content-Type: application/json"

>> Cookie: Sends stored cookies to the server.
	Example: "Cookie: sessionId=sessionTokenHere"

>> Host: Specifies the domain name of the server (for virtual hosting).
	Example: "Host: www.website.com"

>> User-Agent: Identifies the client software.
	Example: "User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64)"

>> Referer: URL of the resource from which the request originated.
	Example: "Referer: https://www.website.com/"



```



# Response Headers
```python
>> Access-Control-Allow-Origin: Specifies the domain that can access the resource.
	Example: "Access-Control-Allow-Origin: *"

>> Cache-Control: Directives for caching mechanisms in both requests and responses.
	Example: "Cache-Control: max-age=3600"

>> Content-Encoding: Specifies any encodings applied to the body (e.g., gzipA).
	Example: "Content-Encoding: gzip"

>> Content-Length: Size of the response body in bytes.
	Example: "Content-Length: 1640"

>> Content-Type: Indicates the media type of the response body.
	Example: "Content-Type: text/html; charset=UTF-8"

>> Set-Cookie: Sends cookies from the server to the client.
	Example: "Set-Cookie: sessionId=h4k56h4gv3vb5jk6; HttpOnly"

>> Expires: Date/Time after which the response is considered stale.
	Example: "Expires: Wed, 21 Oct 2023 07:28:00 GMT"

>> Last-Modified: Date/time of last modification of the resource.
	Example: "Last-Modified: Tue, 15 Nov 2022 12:45:23 GMT"

>> Location: Redirects the client to a different URL.
	Example: "Location: https://www.website.com/newpage"

>> Server: Identifies the software used by the server.
	Example: "Server: Apache/2.4.1 (Unix)"

>> Strict-Transport-Security: abbreviated as HSTS, informs browsers that the site should only be accessed using HTTPS, and that any future attempts to access it using HTTP should automatically be converted to HTTPS.
	Example: "Strict-Transport-Security: max-age=<expire-time>"
```
