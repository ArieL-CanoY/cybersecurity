# HTTP Methods

#### Get Requests
```json
This is used for getting information from a web server
```


#### POST Requests
```json
This is used for submitting data to the web server and potentially creating new records
```


#### PUT Requests
```json
This is used for submitting data to a web server to update information.
Note: All the required data should submitted even though updating only some part of information
```


#### PUT Requests
```json
This is used for submitting data to a web server to update information.
Note: Only the information that will update should be on the body and not all other information.
```



#### DELETE Requests
```json
This is used for deleting information/records
```
