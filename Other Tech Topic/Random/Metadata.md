- data or information that describes about a data.

Example:

> Descriptive Metadata
	>> provides information to identify and describe a resource.
	>> ex. in file, these are file size, file name, file extension, date created and mofidied, etc.

> Structural Metadata
	>> describes how different components of a resource are organize and related .
	>> ex. table of contents, page numbers, sections, chapters, and data schemas.

> Administrative Metadata
	>> provides information to manage a resource.
	>> ex. rights metadata - copyright status, licensing terms, usage restrictions.
	>> preservation metadata - file integrity checksums, migration history, etc.

> Provenance Metadata
	>> details about the history and origin of a resource.
	>> ex. creation history, source of data, transformation applied, and ownership changes.
