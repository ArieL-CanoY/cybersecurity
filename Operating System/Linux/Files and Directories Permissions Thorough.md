# Directories

```python
directory == folder

--- r    - can read the directory like using ls command

--- w    - can create folder and files to the directory

--- x    - can change the current directory to that directory
```

# Files

```python
--- r    - can read the file, can use by cat command, can execute by using command based on the executable. If it is bash file, it can execute using bash command even it is only read permission

--- w    - can edit and overwrite or append text on the file. If the file is only write permission, it cannot read, it cannot write using text editor but it can overwrite or append text

--- x    - can execute the file
```
