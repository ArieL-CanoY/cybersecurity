# curl
```python
curl [ip address/web address]:[port number] - get the content of the website and output to the command line
	options:
		-i                - get the header and body of the response
		-I                - get only the header of the response
		-L                - follow redirect
		-o [filename]     - output the filename
		-u <user>:<pass>  - used when website has a basic authentication
		-v                - verbose mode or enable to throw more information


ex.
	curl https://www.google.com:443 -v 

```
