 - note that it is not checking the data or return type, it is only for readability of programmer of what the data or return type should be.

# Data type hint
```python
name: str = 'aiden pearce'
age: int = 28
isAdult: bool = True

print(name)
print(age)
print(isAdult)

```




# Return type hint
```python
def add(num1: int, num2: int) -> int:
    return num1 + num2

result = add(5, 3)
print(result)

```
