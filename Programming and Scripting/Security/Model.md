


# DREAD
```
DREAD is a quantitative assessment framework that evaluates the severity of a threat by assigning a scaled rating to risk. It stands for Damage, Reproducibility, Exploitability, Affected, and Discoverability.
```


# STRIDE
```
STRIDE is a qualitative threat modeling framework that identifies potential security threats by categorizing them into six categories. It stands for Spoofing, Tampering, Repudiation, Information Disclosure, Denial of Service (DoS), Elevation of Privilege.
```
